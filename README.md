# AI-Ментор: главная страница

Готовый фронтенд-прототип главной страницы AI-Ментор по требованиям (сайдбар, каталог, аналитика, роли, поиск/фильтры/сортировка, сохранение состояния).

## Файлы
- `index.html` - layout и основные секции
- `styles.css` - визуальный стиль и адаптив
- `app.js` - ролевые ограничения и поведение интерфейса

## Быстрый запуск
Откройте `index.html` в браузере или поднимите статический сервер из каталога проекта.

Пример через Node.js:
```bash
npx serve .
```

## Интеграция с Альфа-Курс
Страница поддерживает внедрение данных через глобальный объект `window.AI_MENTOR_BOOTSTRAP` до подключения `app.js`.

Пример:
```html
<script>
  window.AI_MENTOR_BOOTSTRAP = {
    currentUser: {
      id: "u-999",
      name: "Иван Смирнов",
      roleName: "Администратор",
      rights: {
        canAccessHome: true,
        canViewCatalog: true,
        canViewAnalytics: true,
        canCreate: true,
        isAdmin: true,
        allowedUnitIds: []
      }
    },
    units: [
      {
        id: "edu-101",
        title: "Пример обучения",
        type: "Обучающая",
        category: "Коммуникации",
        factory: "КЦ",
        authorId: "u-999",
        authorName: "Иван Смирнов",
        createdAt: "2026-03-01T10:00:00Z",
        updatedAt: "2026-03-10T09:00:00Z",
        launchUrl: "https://alpha-course.example/learn/edu-101",
        editUrl: "https://ai-mentor.example/builder/edit/edu-101"
      }
    ],
    createUrl: "https://ai-mentor.example/builder/create"
  };
</script>
<script src="./app.js"></script>
```

## Что реализовано по требованиям
- Левый сайдбар: `Каталог обучения`, `Аналитика`, активный раздел, условное отображение по правам
- Центральная рабочая зона: каталог по умолчанию
- Кнопка `Создать обучение`: в правом верхнем углу, скрывается без права `canCreate`
- Каталог карточек: открыть обучение (новая вкладка), редактировать только автору/администратору
- Навигация каталога: поиск по названию, фильтры (тип, категория, фабрика), сортировка
- Совместное применение параметров + `Сбросить`
- Пустые состояния: нет доступных данных / нет результатов
- Сохранение состояния каталога между разделами и при возврате через `sessionStorage`
- Ограничение входа на главную по праву `canAccessHome`
